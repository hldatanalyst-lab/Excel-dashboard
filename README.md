# Excel Sales Dashboard

This project contains an interactive **Excel Sales Dashboard** built using  
Pivot Tables, Slicers, Charts, Conditional Formatting, and KPIs.

## Features
- Sales analysis by region, category, and month  
- Dynamic slicers for filtering  
- KPI indicators for total sales & YoY trends  
- Visually clean dashboard for business review
